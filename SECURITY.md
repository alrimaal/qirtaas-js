# Security Policy

## Reporting a vulnerability

Please report vulnerabilities privately via GitHub's
[private vulnerability reporting](https://github.com/alrimaal/qirtaas-js/security/advisories/new)
on this repository — do **not** open a public issue.

Reports affecting the hosted Qirtaas service (api.qirtaas.io / qirtaas.io) are
also in scope and can be filed the same way.

We aim to acknowledge reports within a few business days. Please include a
minimal reproduction where possible.

## Supported versions

Only the latest published version of each `@qirtaas/*` package receives
security fixes.
